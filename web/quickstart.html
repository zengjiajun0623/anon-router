Quickstart

Private AI inference in about two minutes. Fund once, then point any OpenAI- or Anthropic-compatible tool at a local endpoint. No account, no card, no code changes.

Install → Get credit → Start the proxy → Call it

  1. 1

    Install the CLI

    Needs Python 3.10+. A virtualenv always works (macOS often has no bare pip):

    python3 -m venv anon-env
source anon-env/bin/activate
pip install git+https://github.com/zengjiajun0623/anon-router.git
    Other ways to install

    One global command, no venv to activate, with pipx or uv:

    pipx install git+https://github.com/zengjiajun0623/anon-router.git
# or
uv tool install git+https://github.com/zengjiajun0623/anon-router.git

    A published pip install anon-router is coming.

  2. 2

    Get credit

    Got a voucher code? This is the easy path, no crypto:

    anon-router redeem ar-XXXXXXXX

    Mints spendable credit right away. That's it. On to step 3.

    Or deposit testnet ETH instead

    Deposit Sepolia test ETH from a funding key file (never paste a raw key on the command line), then drain the whole balance to ecash:

    anon-router deposit 0.05 --key wallet.json
anon-router claim

    Need test ETH? Free from a faucet (Google Cloud or pk910), sent to your funding wallet's address.

  3. 3

    Start the proxy

    anon-router serve

    Serves an OpenAI + Anthropic API at http://127.0.0.1:8788. Leave it running. Add --tor to also hide your IP from the router.

  4. 4

    Make a call

    Quickest check, from another terminal:

    curl http://127.0.0.1:8788/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{"model":"openai/gpt-4o-mini","messages":[{"role":"user","content":"say hi"}]}'

    ✓ You'll get a normal OpenAI-style JSON reply, paid privately with ecash and unlinkable to your deposit.

Use it from your tools

It's a drop-in endpoint. Base URL http://127.0.0.1:8788/v1, API key = any non-empty string.

OpenAI SDK · Python
from openai import OpenAI
client = OpenAI(
  base_url="http://127.0.0.1:8788/v1",
  api_key="anon")
r = client.chat.completions.create(
  model="openai/gpt-4o-mini",
  messages=[{"role":"user","content":"hi"}])
print(r.choices[0].message.content)
Claude Code · Anthropic
export ANTHROPIC_BASE_URL=http://127.0.0.1:8788
export ANTHROPIC_API_KEY=anon
claude

Cursor, aider, LangChain, LiteLLM: anything OpenAI-compatible works with the same base URL and any key.

Models

Use provider-prefixed IDs like openai/gpt-4o-mini or anthropic/claude-3.5-sonnet; bare names (gpt-4o-mini) are mapped for you; local/* is free. List everything with anon-router models. Billing rounds to a 1-credit ($0.0001) minimum per request.

What's private, honestly

Your spends are cryptographically unlinkable to your deposit (blind-signed ecash), and the model provider only ever sees the router, never you. Limits, plainly: the model reads your prompt (use a local/* model to keep it off third parties); your balance is custodial (keep it small); on-chain funding is pseudonymous (fund from a fresh wallet, a common round amount); and without --tor your IP and timing can still correlate your requests.
